#### nodejs及包管理使用
1. 安装node和npm
    brew install node

2. 安装express
    npm install -g express
    npm install -g express-generator      express版本4以上需要安装
    npm install -g node-dev    安装node-dev
    npm run dev			启动后每次修改会自动重启
